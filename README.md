# Smart Library Request Workflow in ServiceNow

## Project Description
This project is developed using ServiceNow platform to manage library requests efficiently.

## Features
- Role Creation (Librarian, Student)
- Table Creation (Books, Requests)
- Flow Designer Automation
- UI Policy Implementation
- Reports Generation

## Tools Used
- ServiceNow Developer Instance
- Flow Designer
- UI Builder

## Outcome
The system allows students to request books and librarians to manage approvals efficiently.
